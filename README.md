# Code-Basic-part--8

This repository contains multiple beginner-level C++ programs designed to build a strong foundation in programming fundamentals. Each program demonstrates a key concept such as functions, loops, arrays, structures, and basic calculations.

## Included Programs
- Find the maximum of two numbers
- Calculate the sum of natural numbers
- Compute the area of a circle
- Sum all elements in an array
- Check whether a year is a leap year
- Store and display student information using structures
- Store employee data and find the highest salary

## Purpose
These programs help new learners understand:
- Input/output handling  
- Function usage  
- Control structures  
- Arrays and iteration  
- Struct-based data modeling  

## How to Run
1. Copy any program into a `.cpp` file.
2. Compile using a C++ compiler:
   g++ filename.cpp -o output
3. Run:
   ./output
